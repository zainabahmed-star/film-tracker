# Film Tracker 

Film Tracker is a full-stack website that helps movie fans track the films they've watched and want to watch.
Users can browse a shared movie catalog, add new movies, mark them as watched/watchlist, and add reviews.

## Getting Started 
- Deploy link:
- https://trello.com/b/6v9QGA0l/movie-project

### How To Use 
- Sign up for an acoount or sign in.
- Browse the movie catalog, or add a new movie of your own.
- Mark a movie as "Watched" or "Watchlist".
- Leave a review and rating for any movie.
- Visit your watched & watchlist pages to manage your lists.
